## About Me

I develop robotics software with a strong emphasis on control systems, physics-based simulation, and real-world applications. My work focuses on creating intuitive, accessible tools that lower the barrier to entry for robotics, along with well-documented projects that are easy for others to learn from, install, reproduce, and build upon.
